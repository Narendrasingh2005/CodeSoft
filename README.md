## ✨ Features
- 📝 Enter marks for multiple subjects  
- ➕ Calculates **Total Marks**  
- 📊 Calculates **Average Percentage**  
- 🏆 Assigns **Grade (A+, A, B, C, D, F)** automatically  
- 🎨 Attractive and responsive frontend  
- ⚡ Powered by Spring Boot REST API
